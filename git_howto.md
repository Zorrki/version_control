# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

# Работа с удаленными репозиториями
## Проверка какой задан удаленный репозиторий
```sh
git remote show
```
## Более подробно можно посмотреть какой репозиторий указан на чтение (fetch) и запись (push)
```sh
git remote -v
```
